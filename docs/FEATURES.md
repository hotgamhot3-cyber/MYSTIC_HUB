# Features Overview\n\n## 1. Authentication System\n\n### Google Login\n- OAuth 2.0 integration\n- Automatic profile creation\n- Cross-platform support\n\n### Discord Login\n- Discord OAuth integration\n- Discord community linking\n- Custom authentication flow\n\n### Guest Mode\n- Anonymous authentication\n- Limited access to features\n- Upgrade to full account anytime\n\n**Implementation**: `lib/presentation/pages/auth/auth_page.dart`\n\n---\n\n## 2. Home Screen\n\n### User Statistics\n- **Mystic Points**: Total points earned\n- **Level**: Current player level\n- **XP Progress**: XP bar towards next level\n- **Rank**: Global leaderboard position\n\n### Daily Rewards\n- Claimable once per day\n- 500 Mystic Points reward\n- Countdown timer\n- Streak tracking (future)\n\n### Quick Actions\n- Play Game button\n- Academy access\n- Meme Realm\n- Profile view\n\n### Leaderboard Preview\n- Top 3 players\n- User's current position\n- Points display\n\n**Implementation**: `lib/presentation/pages/home/home_page.dart`\n\n---\n\n## 3. Tap-to-Earn Mini Game\n\n### Game Mechanics\n- **Magical Orb**: Central interactive element\n- **Tap to Earn**: Each tap = 10 points × combo multiplier\n- **Combo System**: \n  - Starts at 1x\n  - Increases with each consecutive tap\n  - Maximum 10x multiplier\n  - Resets after 2 seconds of inactivity\n\n### Features\n- Real-time scoring\n- Floating text animations\n- Audio feedback\n- Smooth animations\n- Max combo tracking\n\n**Implementation**: `lib/presentation/pages/tap_game/tap_game_page.dart`\n\n---\n\n## 4.
